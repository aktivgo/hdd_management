# hdd_management
Лабораторная 1 по Системному программированию

## Приложение обладает следующим функционалом:
### 1) тип диска;
### 2) метка тома;
### 3) файловая система;
### 4) серийный номер;
### 5) размер диска в Гб;
### 6) размер занятого пространства в Гб;
### 7) размер свободного пространства в Гб.
